# Mukesh Video Editor - Professional Website

A professional website for Mukesh, a video editor specializing in cricket gear content creation.

## 🌟 Features

- **Modern, Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Layout**: Clean, modern design that showcases video editing services
- **Cricket Gear Focus**: Specifically tailored for cricket equipment content creation
- **Interactive Contact Form**: Professional contact form with validation
- **Instagram Integration**: Links to Instagram page throughout the site
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## 📄 Pages

1. **Home (index.html)**: Landing page with hero section, services, portfolio, and statistics
2. **About (about.html)**: Detailed information about Mukesh's background, skills, and expertise
3. **Contact (contact.html)**: Professional contact form and contact information

## 🎨 Design Features

- Beautiful gradient backgrounds
- Smooth animations and transitions
- Font Awesome icons

- Professional color scheme (blue and purple gradients)

## 🛠️ Technologies Used

- HTML5

## 🚀 Getting Started

### Option 1: Simple File Opening
1. Download all files to a folder
2. Open `index.html` in your web browser

### Option 2: Local Server (Recommended)
1. Navigate to the project folder in terminal
2. Run: `python3 -m http.server 8000`
3. Open: `http://localhost:8000` in your browser
